Docker: -> devops tool
    a virtualization software that makes developing an deploying apps much easier by packaging the application into a container
    with all the neccessary dependancies(i.e npm, djangfo etc), configurations, system tools, and runtime 
    the package can then be easily shared and it is more portable and distributed


what problem Docker solves:
    while developing an application before containers the developeers would have to install all the services their application
    depnds on or need directly on their os on their machine 

    but with container we do not have to install all the services direcctly on our sysytem 
    with containers we package verything we need toa a container

    and solves many other problems

    environment reproducibility: every one gets tye same copy and exact setup
    dependancy management: no langs hell, os specific issues etc
    portability: run your container on any machine with docker(dev laptop, server or cloud )
    version control for environment: docker file are text = versionable in git
